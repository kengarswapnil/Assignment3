# DeepThought Node.js CRUD API Assignment

## Project Overview
This project is a Node.js CRUD API for managing users, built using **Express.js** and **MongoDB**.  
It is developed as part of **DeepThought Node.js Internship Task** to demonstrate the ability to read requirements, implement backend APIs, and document them.

The API allows you to:
- Create a user
- Read all users or a single user
- Update a user
- Delete a user

---

## Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB with Mongoose ODM  
- **Environment Management:** dotenv  
- **Others:** Postman or Insomnia for testing
